================================================================================
KEY FEATURES
================================================================================

1. HTML PARSING
   - Extract forms and input fields
   - Find all script tags (inline and external)
   - Detect template variables (Jinja2, Django, ERB, EJS, Angular)
   - Identify dangerous HTML attributes
   - Parse meta tags and security headers

2. FORM SCANNER
   - Detect missing CSRF tokens
   - Warn on GET method forms
   - Identify fields without validation
   - Check autocomplete settings
   - Generate test payloads

3. OUTPUT SCANNER
   - Find unescaped template variables
   - Detect dangerous contexts (script, style, URLs)
   - Check for user input in execution contexts
   - Identify string concatenation patterns
   - Verify output encoding

4. JAVASCRIPT SCANNER
   - Detect dangerous functions (innerHTML, eval, etc.)
   - Find DOM manipulation with user input
   - Check postMessage handlers
   - Identify localStorage usage
   - Detect location object manipulation

5. HTTP SCANNER
   - Test for reflected XSS in GET/POST parameters
   - Analyze security headers
   - Check cookie security flags
   - Generate educational sandbox page

6. REPORT GENERATOR
   - HTML reports with severity color-coding
   - Plain text reports
   - JSON reports for CI/CD
   - Summary statistics by severity
   - Remediation recommendations

7. KNOWLEDGE BASE
   - XSS type definitions (Reflected, Stored, DOM-based)
   - Test payloads for each type
   - Prevention techniques
   - Dangerous function alternatives
   - Escaping rules for all contexts
   - Security header recommendations

================================================================================
VULNERABILITIES DETECTED
================================================================================

1. InnerHTML Usage
   - Detects element.innerHTML = userInput
   - Suggests textContent alternative

2. Template Variables
   - Finds {{ user_input }} without escaping
   - Suggests {{ user_input|escape }}

3. Inline Event Handlers
   - Finds <img onerror=alert(1)>
   - Suggests addEventListener

4. eval() Function
   - Detects eval(userInput)
   - Suggests JSON.parse()

5. document.write()
   - Detects document.write with user input
   - Suggests createElement

6. setTimeout with Strings
   - Detects setTimeout('code', 1000)
   - Suggests setTimeout(() => {}, 1000)

7. Missing CSRF Tokens
   - Detects forms without csrf_token
   - Suggests adding hidden input

8. Unescaped Output in Scripts
   - Finds var data = "{{ user_input }}"
   - Suggests var data = {{ user_input|tojson }}

9. href with javascript:
   - Detects <a href="javascript:...">
   - Suggests proper event handling

10. DOM-based XSS
    - Detects element.innerHTML = location.hash
    - Suggests sanitization

================================================================================
PREVENTION TECHNIQUES
================================================================================

1. OUTPUT ENCODING
   - HTML context: < > & " '
   - JavaScript context: backslash escaping
   - URL context: percent encoding
   - CSS context: backslash escaping

2. CONTENT SECURITY POLICY
   - default-src directives
   - script-src restrictions
   - object-src limitations

3. SECURE CODING
   - textContent vs innerHTML
   - createElement vs HTML strings
   - JSON.parse vs eval

4. SECURITY HEADERS
   - X-Frame-Options
   - X-Content-Type-Options
   - Strict-Transport-Security

5. COOKIE SECURITY
   - HttpOnly flag
   - Secure flag
   - SameSite attribute
